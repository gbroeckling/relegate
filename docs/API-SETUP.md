# API Setup Guide

Relegate uses the same API integrations as [GideonMail](https://github.com/gbroeckling/gideonmail). For detailed step-by-step instructions with screenshots, see the [GideonMail API Setup Guide](https://github.com/gbroeckling/gideonmail/blob/master/docs/API-SETUP.md).

## Required

### IMAP/SMTP Email Server
Any standard IMAP server: ISPConfig, Dovecot, Postfix, Zimbra, etc.

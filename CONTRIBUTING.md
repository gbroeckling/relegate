# Contributing

Thanks for your interest in Relegate.

## Current Status

Relegate is in pre-release development. The core features are being built and tested in [GideonMail](https://github.com/gbroeckling/gideonmail), the desktop companion app.

## How to Help

- **Test features** — Install [GideonMail](https://github.com/gbroeckling/gideonmail) and try the AI triage, SMS alerts, calendar integration, and sender management
- **Report bugs** — File issues against GideonMail for now; they'll be fixed in both
- **Request features** — Open an issue here describing your use case

## Development Philosophy

Built by **Garry Broeckling**. Implementation is AI-assisted using Claude by Anthropic — all architecture, product decisions, testing, and releases are human-directed. Contributors are welcome to use any tools they prefer.
